La etiqueta hr cambio temático entre párrafos

Listas

Las etiquetas ol y ul definen listas de elementos. La diferencia entre ellas es que, en la etiqueta ol, la lista definida será de manera ordenena, mientras que con la etiqueta ul, la lista no tendrá orden. A cada elemento de las listas se definen con la etiqueta li.
Veamos los siguientes ejemplos, primero estará una lista ordenada, seguido de una lista son orden.

  1. Módulo A
  2. Módulo B
  3. Módulo C
  4. Módulo D
  • Módulo 1
  • Módulo 2
  • Módulo 3
  • Módulo 4
